# Demo slides (HTML)

Paste your slide HTML into the files in this folder:

- **slide-1.html** — first slide
- **slide-2.html** — second slide
- **slide-3.html**, **slide-4.html**, … — add more files; order is by filename

Use plain HTML (e.g. `<h1>`, `<p>`, `<ul>`, `<a href="...">`, inline `style="..."`). Content is rendered inside the slide layout; no need for `<html>` or `<body>`. Only trusted content — these files are loaded at build time and rendered as-is.
